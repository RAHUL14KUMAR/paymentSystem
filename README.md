## How To Start The System

## 1-> clone the repository

`git clone https://github.com/RAHUL14KUMAR/paymentSystem.git`

## 2-> install the dependecies

run- `npm install`


## 3-> add .env file in the root directory

takes 2 things

`MONGO_URI` AND `SECRET_KEY`

##4-> start the server

run `npm start`
